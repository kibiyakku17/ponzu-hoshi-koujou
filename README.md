# ひらけ！ほしこうじょう

ぽんず工房のスマホ向け2Dサンドボックスゲーム。

未知の惑星に不時着した探査ロボが、資源を採って工場を組み、
通信アンテナを建てて救難信号を送るまでを目指す。

- **プレイ**: https://kibiyakku17.github.io/ponzu-hoshi-koujou/
- 構成: `index.html` 単一ファイル（HTML/CSS/JS全部入り・外部依存なし）
- セーブ: localStorage 自動保存（10秒ごと）＋ JSONエクスポート/インポート（⚙️設定）

## あそびかた

1. タップで移動、鉱脈タップで採掘（鉄鉱石・石・エネルギー結晶）
2. 🔨建設 → 精錬炉（鉄鉱石→鉄板、石→石ブロック）
3. 作業台で部品をクラフト（鉄板×2）
4. 採掘機を鉱脈に置いて自動化（隣の保管箱/精錬炉へ自動受け渡し）
5. 通信アンテナ（部品×5＋石ブロック×5＋結晶×3）を拠点付近に建てて起動 → クリア！

バッテリーに注意。拠点の近くで自動充電、切れると強制帰還（資源の一部を失う）。

## 調整パラメータ

バランス調整は `index.html` 冒頭の `CONFIG` オブジェクトにまとまっている
（産出速度・精錬時間・設置コスト・バッテリー消費など）。

## 回帰テスト

外部パッケージなしで、埋め込みJavaScriptの構文・地形生成・留守中生産・セーブ移行を確認できる。

```powershell
node tests/regression.cjs
```
